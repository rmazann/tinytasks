# TinyTask

A dead simple task management application built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Add new tasks
- Toggle task completion
- View completed tasks (collapsible section)
- Clean, minimalist design matching the Figma design

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18
